Web Spreadsheet

Supporting insert, delete, copy and clear

Built a web spreadsheet in from server to browser using NodeJS, ExpressJS, Mustache temmplating framework, HTML5, CSS and ReactJS.

Starting with the backend, I modeled the spreadsheet as a directed acyclic graph with dependencies to as to emulate the functionality of adding values to a cell, adding formulae to a cell, copying values and formulae of te cells, deleting the value of a particular cell and clearing the spreadsheet.

Building on that, the spreadsheet functionality was made to persist by using MongoDB as the database and making asynchromous calls to the database.

Then, I added web routes for each of the functionalities of insert, copy, delete and clear using ExpressJS

Using Mustache, HTML5 and CSS, create the front end for the web spreasheet similar to the earlier style of web programming.Finally, the modern Javascript framework, ReactJS, was used for building the front end.
