Optimized Delivery System

For efficient package deliveries

Given a set of N delivery spots and distance between each pair of delivery spots, I worked on finding the shortest possible optimal path, minimising travel cost between each pair of delivery spots, with each spot visited exactly once. This problem, which closely relates to Traveling Salesperson Problem, was solved using dynamic programming and backtracking.

The main idea behind dynamic programming was to compute the optimal solution for all subpaths of length N while using data from the already known optimal partial tours of length N-1. The set of visited nodes in the subpath and the index of the last visited node in the path, these two form the dynamic programming state.

In the backtracking approach, the algorithm recursively solved for the optimal path, building the solution in increments, removing the solutions that fail to satisfy the constraint for travelling salesman problem, minimising cost of travelling from start spot, visiting all spots and returning to the start spot.
